# 📄 CV en Ligne - Amin BAHADDI

> Version moderne et interactive de mon Curriculum Vitae, optimisée pour le web.

Ce projet présente mon parcours professionnel, mes compétences techniques et mes projets en tant qu'**Ingénieur DevOps et Cloud**. Il intègre une interface utilisateur épurée ainsi qu'une fenêtre contextuelle interactive (popup) développée en JavaScript natif.

## 🚀 Démo en Ligne

Le projet est hébergé et accessible en direct à l'adresse suivante :
👉 **[Consulter mon CV en ligne](https://abahaddi.github.io/My_CV_Project/)**

---

## 🛠️ Technologies Utilisées

*   **HTML5** : Structure sémantique du contenu.
*   **CSS** : Design moderne, mise en page double colonne adaptative (Flexbox) et variables de styles.
*   **JavaScript** : Logique d'interaction pour la fermeture dynamique de la popup d'accueil (sans dépendance externe).
